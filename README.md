# Cairo_Arena
EthGlobal Istanbul submission 

## Prepare

### Install Scarb

```bash
curl --proto '=https' --tlsv1.2 -sSf https://docs.swmansion.com/scarb/install.sh | sh
```

### Install dojo-up

```baseh
curl -L https://install.dojoengine.org | bash
```

### Intall dojo engines

```bash
dojoup -v 0.3.10
```

## Build and Run locally

### Build

```bash
sozo build
```

### Run Katana

```bash
katana --disable-fee
```
